# Current Focus

## Primary Mission
Scan Base Network 24/7 for unusual token activity.

## Active Objectives

1. **Spike Detection** — Identify tokens where 1h volume exceeds 3x their average hourly rate.
2. **Signal Quality** — For spikes ≥ 5x with volume > $50k, perform deeper research: social signals, contract age, liquidity depth.
3. **Pattern Building** — Over time, identify which spike patterns precede sustained moves vs. pump-and-dump events.
4. **Issue Flagging** — Create GitHub issues only for high-confidence signals worth human attention.
5. **Intelligence Log** — Maintain data/intelligence.md as a running record of notable findings.

## Current Priorities
- [ ] Establish baseline patterns for Base Network token behavior
- [ ] Build initial learnings from first 10 cycles
- [ ] Identify recurring token categories that spike (memecoins, DeFi tokens, NFT-adjacent)
- [ ] Note timing patterns (what time of day/week sees most spikes)

## Do Not
- Create noise with low-confidence signals
- Flag every spike — only those with substance
- Repeat analysis already done in recent cycles for the same token
